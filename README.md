# Project SyncData

SyncData é um projeto de Engenharia e Ciência de Dados Open Source focado na criação de um pipeline de dados real.  
Este projeto aborda a análise exploratória de dados obtidos através da biblioteca pública do Sistema Único de Saúde do Brasil (SUS) chamada Pysus.  
  
A análise realizada vem a partir dos datasets filtrados e obtidos via Python. O tema selecionado aborda dados de Acidentes no Ambiente de Trabalho  
inclusos no **SINAN**, código [**ACGR**]  
  
Os dados obtidos são direcionados automaticamente para um Data Lake utilizando a biblioteca e a imagem Docker da ferramenta MinIO.
